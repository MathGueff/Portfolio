# Portfólio – MathGueff

Portfólio em Next.js e React com tema **Lua e Animais** (cores noturnas, âmbar e toques de natureza).

## Conteúdo

- **Hero:** apresentação com avatar (GitHub), nome, título e localização
- **Sobre:** formação Fatec Votorantim, experiência e foco em estágio
- **Tecnologias:** Java, Angular, TypeScript, HTML5, CSS, Bootstrap, JavaScript, Git, Node.js, Sequelize, Figma, SQL Server
- **Projetos:** SaneaSP (Front-end e Back-end) e PvZ-Adventures-Fanmade com links para repositórios
- **Contato:** GitHub, LinkedIn e e-mail

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Estrutura

- `app/` – layout, página inicial, estilos globais e ícone (favicon)
- `components/layout/` – Header (navegação por âncoras) e Footer
- `components/sections/` – Hero, About, Technologies, Projects, Contact
- `lib/data.ts` – dados do perfil, projetos, tecnologias e links (edite aqui para atualizar o conteúdo)
